## 使い方

> [!NOTE] 
>
> ### 前提条件
> 
> - [ ] Visual Studio Code のインストール
> - [ ] Git のインストール
> - [ ] Docker のインストール
> - [ ] VSCodeの拡張機能 "Dev Containers" のインストール

1. 対応言語・環境のディレクトリをコピーする
2. VSCodeでコピーしたディレクトリを開く
3. `Ctrl + Shift + P` (Macの場合は `Cmd + Shift + P`) を押してコマンドパレットを開く
4. 「Dev Containers: Reopen in Container」を選択する
5. DevContainerが立ち上がるのを待つ
6. 開発を始める
